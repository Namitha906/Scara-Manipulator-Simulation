# 3-DOF SCARA Manipulator Simulation

This project demonstrates the design and simulation of a 3-DOF SCARA manipulator for pick-and-place tasks.

## Tools Used
- MATLAB (kinematics simulation)
- Blender (animation)
- TinkerCAD (robot design)

## Features
- Inverse kinematics for joint angle calculation
- Forward kinematics for robot visualization
- Pick-and-place trajectory simulation
- 3D animation of robot motion

## Files
- MATLAB code for simulation
- Blender animation video
- Project report
